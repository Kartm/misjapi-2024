1.  **Developing**

    ```shell
    npm run develop
    ```
    
    You might need to run it a couple of times, as the website sometimes fails to deliver all images.

    Visit http://localhost:8000/__graphql to access the GraphQL playground.

2.  **Building**

    ```shell
    npm run build
    ```
    
    You can test it via `npx serve public`
